# PortfolioHUB - Implantação e Segurança

Plataforma centralizada para gerenciamento e exibição de portfólios digitais, integrada à API do GitHub e otimizada com o auxílio da IA Google GEMINI.

## 🛠️ Tecnologias e Boas Práticas
- Controle de versão via Git/GitHub.
- Gerenciamento de credenciais seguro por variáveis de ambiente.
- Governança de código com regras de proteção de branch.
